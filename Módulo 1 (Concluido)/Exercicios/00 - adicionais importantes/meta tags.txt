Adicionais e complementos:


<meta http-equiv="refresh" content="seconds; url=URL">: Redireciona automaticamente o navegador para outra página após um determinado número de segundos.

<meta http-equiv="pragma" content="value">: Define a política de cache para a página. Os valores comuns incluem no-cache, no-store, public, private, etc.

<meta http-equiv="cache-control" content="value">: Similar a <meta http-equiv="pragma">, define a política de cache para a página.

<meta http-equiv="expires" content="value">: Define a data de expiração do cache para a página.

<meta name="referrer" content="value">: Controla o comportamento do cabeçalho Referer quando os usuários seguem links para fora da página. Os valores comuns incluem no-referrer, origin, origin-when-cross-origin, same-origin, etc.

<meta name="format-detection" content="value">: Controla a detecção automática de formatos de números de telefone e endereços na página. Os valores comuns incluem telephone=no, address=no.
